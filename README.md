
## Setup

First, clone the repo via git:

```bash
git clone --depth=1 git@github.com:bvluong/music_player.gityour-project-name
```

And then install dependencies with npm:

```bash
$ cd your-project-name
$ npm install
```

To start the app:

```bash
$ npm run webpack
```

```bash
$ open index.html
```
